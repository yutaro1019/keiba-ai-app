# keiba-ai-app

競馬AIの予想、買い目提案、シミュレーション、情報更新をWeb画面から使えるアプリです。

## モデル・予想精度

過去データで確認している主な参考指標です。

| 指標 | 数値 |
|---|---:|
| 学習データ | 2017〜2024年 |
| 検証データ | 2025年 |
| TOP3アンサンブル AUC | 0.8167 |
| WINアンサンブル AUC | 0.8370 |
| 予想1位の1着的中率 | 33.2% |
| 予想1位の3着内率 | 63.2% |
| 予想上位3頭中の3着内平均 | 1.54頭/レース |
| ハイブリッド回収率（2025年・実払い戻しベース） | 111.1% |

実行条件やデータ更新状況によって数値は変わります。Webアプリのシミュレーション結果を優先して確認してください。

## 推奨モード

現在のおすすめは **ハイブリッド** です。

| モード | 買い方の考え方 | 特徴 |
|---|---|---|
| ハイブリッド | ワイド、三連複、BOX系を候補にして期待値で選択 | 的中率と回収率の両方を見て、三連単のような外れ値に寄りすぎない |
| 的中ハイブリッド | 予想1位・2位の複勝とワイドを土台にして、単勝と三連複を少額追加 | 2頭が3着内に来た時にマイナスになりにくい配分。自信度40以上を初期条件にする |

旧モードの「堅実型」「バランス型」「一発狙い型」「AIおまかせ」は現在のWeb画面では使わない前提です。

## 主な機能

- 未来レース予想
- 日付指定で全レース予想
- 競馬場とR指定で個別レース予想
- 予想順位、自信度、推定勝率、推定3着内率の表示
- 買い目と資金配分の表示
- 期間指定シミュレーション
- 情報更新モード
- 実払い戻しデータを使った検証

## 使い方

```powershell
cd C:\Users\yukim\Downloads\keiba_ai_package\keiba_ai
python -X utf8 src\web_app.py --host 127.0.0.1 --port 7860
```

起動したらブラウザで開きます。

```text
http://127.0.0.1:7860/
```

## テスト

標準ライブラリだけでテストとカバレッジ確認ができます。

```powershell
python -X utf8 tools\run_tests.py --fail-under 80
```

現在のカバレッジ目標は `src/betting.py` で80%以上です。

## 公開しないデータ

スクレイピングで取得したデータは、netkeibaなど取得元サイトの利用規約に配慮し、GitHubでは公開しません。

以下は `.gitignore` で除外しています。

- スクレイピングで取得したレースデータ
- 払い戻しデータ
- オッズデータ
- 学習済みモデル
- ログ
- Pythonキャッシュ

つまり、`data/`、`models/`、`logs/` はGitHubに公開しない前提です。
