# 🏇 KEIBA AI — 競馬予想 & 賭け方提案エンジン

LightGBM アンサンブル × 血統特徴量 × ケリー基準 による本格的な競馬予想 CLI ツール。

## 📊 モデル性能 (2025年データで検証)

| 指標 | 数値 |
|------|------|
| 学習データ | 2017〜2024年 (379,461 行) |
| 検証データ | 2025年 (47,497 行) |
| **TOP3 アンサンブル AUC** | **0.8167** |
| **WIN アンサンブル AUC** | **0.8370** |
| AI予想1位馬の単勝的中率 | **33.2%** |
| AI予想1位馬の複勝的中率 | **63.2%** |
| 予想上位3頭中の3着内平均 | **1.54頭/レース** |

## 🧠 モデル構成

- **TOP3予測**: LightGBM 3シードアンサンブル (seed=42/7/2024、leaves=63/95/47)
- **WIN予測**: LightGBM 2シードアンサンブル (seed=42/7、leaves=63/95)
- **特徴量**: 34個
  - 数値(25): 馬番・枠番・斤量・馬体重・距離・オッズ・人気・過去勝率・過去複勝率・タイム指数・上がり・前走間隔・騎手成績・調教師成績・距離適性・コース適性
  - カテゴリ(9): 性別・コース・回り・天候・馬場・クラス・競馬場・**父馬**・**母父馬**

## 🎯 賭け方スタイル

| スタイル | 主な券種 | 特徴 |
|---------|---------|------|
| 🛡️ 堅実型 | 複勝 + ワイド | 的中率重視、資金を守る |
| ⚖️ バランス型 | 単勝・複勝・馬連・三連複 | バランスの取れた配分 |
| 💥 一発狙い型 | 三連単・馬単・三連複 | 高配当狙い |
| 🤖 AIおまかせ | EV最大組み合わせ | AIが自動で最適券種を選択 |

予算配分は**ケリー基準の分数**(各スタイルで0.25〜0.50)で計算され、100円単位に丸められます。

## 🚀 使い方

### 1) 対話モード(おすすめ)

```bash
cd /home/user/keiba_ai/src
python3 keiba_ai.py
```

→ 「過去レース検証」/「未来レース手入力」/「race_id直接指定」から選択

### 2) コマンドラインモード

```bash
# 過去レースを予想して即賭け方提案
python3 keiba_ai.py --race-id 202506050811 --budget 5000 --style ai

# 未来レース手入力モード
python3 keiba_ai.py --manual
```

### オプション

| オプション | 内容 |
|----------|------|
| `--race-id` | 過去レースの race_id |
| `--budget` | 予算(円、デフォルト3000) |
| `--style` | `kenjitsu` / `balance` / `ippatsu` / `ai` |
| `--manual` | 未来レース手入力モード |
| `--year` | 一覧表示する年(デフォルト2025) |


## ⚠️ 免責事項

- 推定オッズは AI の推測値です。実際のレース直前オッズで微調整してください。
- 推定勝率は LightGBM の予測値の独立近似で、実勝率と乖離する場合があります。
- 競馬は娯楽です。生活費を切り崩しての投資は絶対にお控えください。
